# 使用 uv 虚拟环境运行（推荐）

## 1. 创建虚拟环境（首次）
uv venv

## 2. 安装依赖
uv pip install -r requirements.txt

## 3. 运行 V2 WebUI（Windows）
scripts/start_webui.bat

## 4. 运行 V2 WebUI（macOS / Linux）
./scripts/start_webui.sh

## 5. 或手动启动后端
uv run python main.py --serve-web

## 6. 运行 CLI action
uv run python main.py --action build_symlink_workspace --dry-run
