# EmbyLlmTool - 项目概述

## 项目目标

基于 Emby115Toolkit 的现有功能，引入大模型技术（LLM）进行重构，采用工作流编排方式将 7 个独立功能页串联起来，实现从 "手动逐个操作" 到 "一键自动化执行" 的升级。

## V2 当前命令

推荐动作名：

```bash
python main.py --action build_symlink_workspace
```

兼容别名：

```bash
python main.py --action scan_and_link
```

启动 WebUI 后端骨架：

```bash
python main.py --serve-web
```

## 源项目

- 项目名：Emby115Toolkit
- 路径：C:\dev\Emby115Toolkit
- 当前功能：7个独立的GUI功能页（tkinter + PyQt5双前端）
- 后端模块：autosync/（文件系统操作）、emby/（Emby API操作）
- 配置：YAML单例Config

## V2 代码结构

- `emby115_v2/context.py` — 统一 Context Object
- `emby115_v2/cli.py` — CLI facade
- `emby115_v2/web/` — WebUI backend facade
- `emby115_v2/workflow/` — 工作流执行器
- `emby115_v2/services/` — 核心服务层
- `emby115_v2/reports/` — HTML/JSON 报告生成
