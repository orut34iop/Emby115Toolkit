# EmbyLlmTool - 项目概述

## 项目目标

基于 Emby115Toolkit 的现有功能，引入大模型技术（LLM）进行重构，采用工作流编排方式将 7 个独立功能页串联起来，实现从 "手动逐个操作" 到 "一键自动化执行" 的升级。

## 当前阶段

- [x] 原项目代码深度分析（~6800行Python）
- [x] 竞品UI调研（n8n 2.0、Sonarr/Radarr、Dify、Zapier、Flowise、RJ-Auto-Metadata）
- [x] 功能需求清单（P0-P3分级）
