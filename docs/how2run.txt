# 使用 uv 虚拟环境运行（推荐）

## 1. 创建虚拟环境（首次）
uv venv

## 2. 安装依赖
uv pip install -r requirements.txt

## 3. 运行 tkinter 版本（Windows）
uv run python main.py

## 4. 运行 V2 WebUI（Windows）
scripts/start_webui.bat

或手动启动后端：
uv run python main.py --serve-web

## 5. 或运行 PyQt5 版本（macOS）
uv run python emby115_v1/qt_main.py
